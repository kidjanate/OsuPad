# OsuPad
 A 2k keypad for osu!, I made this keypad with my dad for new year gift
 This repository will tell everything how can I made this.
 
 ### Prepare Stuff
  * 2x 4 cm x 5 cm Acrylic
  * 2x Keyboard switches
  * 2x Keycaps
  * 1x Arduino Pro Micro
  * 2x Short wires
  * 1x Keychain rope
  * 2x LEDs
